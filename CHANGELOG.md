# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Biome for linting and formatting (replaces ESLint + Prettier)
- Vitest testing framework with coverage reporting
- Husky + lint-staged pre-commit hooks
- Docker support (Dockerfile + docker-compose.yml)
- EditorConfig for cross-editor consistency
- Node.js version pinning (.nvmrc)
- Project scaffolding (src/ directory structure)
- CI/CD enhancements (lint check, security audit, Docker build)
- AI standards documentation (security, performance, deployment)
- README, LICENSE (MIT), CONTRIBUTING guide
- Environment variable validation (src/env.ts)

## [0.1.3] - 2026-02-28

### Added

- Initial AI-ready project template
- Hierarchical `.ai/` documentation structure
- Multi-AI tool support (Claude Code, Cursor, Cline, Gemini, VS Code Copilot)
- Task Master AI integration via MCP
- GitHub Actions CI/CD (PR tests, releases, dependabot, branch cleanup)
- TypeScript configuration (strict mode, ES2022)
- Git conventions and code quality standards
