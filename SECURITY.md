## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email **{{SECURITY_EMAIL}}** with:

1. A description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact assessment
4. Any suggested fixes (optional)

### What to expect

- **Acknowledgment** within 48 hours
- **Status update** within 7 days
- **Resolution timeline** communicated based on severity

We follow [responsible disclosure](https://en.wikipedia.org/wiki/Responsible_disclosure) practices and appreciate your help keeping this project secure.
